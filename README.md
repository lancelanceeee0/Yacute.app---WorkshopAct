# 🌸 Yacute App

A cute JavaFX login monitoring system connected with Supabase.

This project was made using:

* JavaFX
* Maven
* Supabase

The app allows users to:

* Log in using a username and password
* Record login timestamps
* View recent login logs
* See a cute mascot after logging in ✨

---

# 💖 Features

* Cute and modern UI
* Login system
* Dashboard page
* Supabase database integration
* Real-time login log storage
* Logout button
* Simple and clean design

---

# 🔐 Default Login

# 🖼️ Dashboard

After logging in, the user can:

* See the Yacute mascot
* View recent login history
* Logout safely

Because every system deserves to be cute too 🌸

---

# ⚙️ Requirements

* Java JDK 21+
* Maven
* Internet connection

---

# 🚀 Run the Project

Open the project in IntelliJ IDEA then run:

```txt
MainApp.java
```

---

# 🌷 Notes

Make sure your `.env` file contains:

```env
SUPABASE_URL=
SUPABASE_ANON_KEY=
```

before running the app.
